# Root-cause mathematics

Formula version: 2.2.0

The application displays all formulas and substitutions in the Root Cause calculation audit. The analysis uses event-relative delta, population standard deviation, event z-score, Pearson correlation, Fisher 95% correlation confidence intervals, p-values, repeated failure rates, and transparent weighted evidence scores.

Evidence scores are rankings, not causal probabilities. A physical root cause still requires supporting RF, installation, maintenance, or hardware evidence.
